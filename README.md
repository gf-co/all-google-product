# Next.js with Supabase

- A React app to demo Supabase's capabilities.
- Using tools with no vendor lock-in.

## Built with

- Next.js
- TypeScript
- Tailwind CSS
- Jest
- Cypress
- MDX (Lets you write pages using Markdown.)
